This package provides a Python3 interface of using
`DTK2 <http://ornl-cees.github.io/DataTransferKit/>`_ and
`MOAB <http://sigma.mcs.anl.gov/moab-library/>`_ in parallel with MPI for
multi-physics coupling problems.
